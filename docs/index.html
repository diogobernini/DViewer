DViewer Sample!

This is a Javascript/jQuery library that allows you to enable document/images visualization by setting the background of the image and changing the position/scale based on user interaction.

Follow the steps below to integrate it into your own webpage!

Dig into the Source

Steps

Call the constructor with the id of the object that will receive the background. Such as a canvas. And the buttons that will interact with it.

var Visualizador = new DViewer("#image", "#btnPlus", "#btnLess", "#btn100", "img1.jpg");

Set the height using 0-100 integer values meaning the percentage of it.

Visualizador.setHeight(50);

If you need to change the image, use:

Visualizador.changeImage("img2.jpg");
